¿Qué es un Proyecto Comunitario?

Es un conjunto de actividades diseñadas para resolver un problema específico de un grupo humano (vecinos, estudiantes, etc.) buscando el bien común.

"No es para beneficio de una sola persona, es para mejorar la vida de TODOS en la comunidad."
